### 프로젝트 개요
학습 목적에 맞게 설빙 페이지를 재설계 및 클론
프로젝트 기간 : 2025-05-16 ~ 2025-05-19
<br>
### 역할 분담 및 기능 구성
|페이지|담당자|주요 기능|
|메인페이지|이석진|메인 배너, 메뉴 섹션, SNS|
|브랜드 페이지|이나영|설빙 스토리, 연혁 BI 소개|
|메뉴 페이지|박경빈, 임나연|전체 메뉴, 카테고리별 메뉴 (설빙, 사이드, 음료)|
|뉴스 & 고객센터|배기열, 이나영|공지사항, FAQ, 온라인 고객센터 상담 내역|
|매장 찾기|이석진|지도 API 연동|
|푸터|임나연|공통 하단 정보|
